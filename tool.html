The Pure Random method is just that - random. You will be assigned units based on the luck of the draw. There is a distinct possibility your Lord will go solo for a while, so you've been warned.

The Point Based Method assigns each unit a value. You will be assigned units until your total value is the selected amount. If you would like to alter this value, or the assigned values of each unit, select the Advanced box below

The Tier List Based Method splits units into tiers based on their value for efficient play You will be assigned a set amount of units from each tier. If you would like to alter this amount, or adjusted the tier placements of the units, select the Advanced box below
